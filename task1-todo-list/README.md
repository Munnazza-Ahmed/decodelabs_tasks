# To-Do List Manager

DecodeLabs – Python Programming Industrial Training Kit
**Project 1: The To-Do List**

A simple command-line to-do list manager built with Python. Demonstrates working with lists, loops, and functions to manage an in-memory task list.

## Features
- Add tasks to your list
- View all current tasks
- Remove a task by its number
- Exit the program

> Note: Tasks are stored in memory only for the current session — the list resets each time you restart the program.

## How to Run
```bash
python ToDo-List.py
```

Then follow the on-screen menu (options 1–4) to manage your tasks.

## Key Concepts Used
- Lists (`append`, `pop`, `enumerate`)
- Functions and separation of concerns (menu, add, view, remove)
- Loops (`while True` main loop, `for` loop to display tasks)
- Basic input validation and error handling (`try`/`except`, empty-input checks)
